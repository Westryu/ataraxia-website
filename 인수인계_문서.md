# Ataraxia 프로젝트 인수인계 문서 (Antigravity용)

이 문서는 새로운 Antigravity 인스턴스가 이 프로젝트를 즉시 파악하고 작업을 이어나갈 수 있도록 돕기 위해 작성되었습니다.

## 프로젝트 개요
- **이름**: Ataraxia (아타락시아)
- **성격**: 운동 센터 또는 크로스핏 박스의 랜딩 페이지 / 웹사이트
- **기술 스택**: React, Vite, JavaScript, CSS (Vanilla CSS)
- **주요 기능**: 메인 Hero 섹션, 여정(Journey), 규칙(Rules), 위치(Location), 와드(WOD), 갤러리(Gallery) 페이지.

## 주요 파일 구조
- `src/App.jsx`: 메인 라우팅 설정.
- `src/components/`: `Navbar`, `Hero` 등 공통 컴포넌트.
- `src/pages/`: 각 메뉴별 페이지 컴포넌트 (`OurRules.jsx`, `OurLocation.jsx` 등).
- `public/ataraxia_bg.mp4`: 메인 배경 영상.

## 최근 작업 내용
- 네비게이션 메뉴 이름 간소화 (RULES, LOCATION, WOD).
- 'Our Rules' 페이지의 이미지 교체 및 디자인 수정.
- 'Gallery' 페이지 추가.
- 모든 스타일은 `src/index.css`에서 중앙 집중식으로 관리됨.

## Antigravity에게 주는 팁
- 이 프로젝트는 고급스러운(Premium) 디자인과 부드러운 애니메이션을 지향합니다.
- 배경 영상은 `public/ataraxia_bg.mp4`를 사용하며, CSS에서 `video` 태그를 통해 처리됩니다.
- 새로운 기능을 추가할 때는 기존 `index.css`의 디자인 토큰을 유지해 주세요.
