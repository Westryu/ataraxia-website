# Ataraxia 이전 가이드

새로운 Google One AI Pro 계정에서 Ataraxia 프로젝트를 설정하는 방법입니다.

## 1단계: 파일 옮기기
1. 현재 이 폴더(`ataraxia-standalone`) 전체를 다운로드하여 로컬 PC에 저장합니다.
2. 새 계정의 Antigravity(또는 개발 환경)에 이 폴더를 업로드하거나 복사합니다.

## 2단계: 의존성 설치
새 환경의 터미널에서 다음 명령어를 실행하세요:
```bash
npm install
```

## 3단계: Antigravity와 대화 시작하기
새 계정의 Antigravity에게 다음과 같이 말하면 프로젝트를 즉시 파악할 수 있습니다:
> "내가 가지고 온 `인수인계_문서.md` 파일을 읽고 이 Ataraxia 프로젝트의 작업을 이어서 해줘."

## 4단계: 로컬 실행 확인
작업을 시작하기 전, 다음 명령어로 사이트가 잘 뜨는지 확인하세요:
```bash
npm run dev
```
